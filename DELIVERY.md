# Руководство по работе с пакетом datagen_delivery

Этот документ описывает порядок использования готового пакета `datagen-go` для генерации синтетических данных. Пакет подготовлен для сотрудников и содержит всё необходимое для работы без необходимости сборки программы из исходного кода.

## 1. Состав пакета

После распаковки архива `datagen_delivery.zip` вы увидите следующие файлы:

- **Исполняемые файлы** (выберите нужный для вашей ОС):
  - `datagen.exe` — для Windows.
  - `datagen_macos` — для macOS.
  - `datagen_linux` — для Linux и Google Colab.
- **Конфигурация**:
  - `config.yaml` — основной файл, где настраиваются параметры генерации.
- **Данные**:
  - `data/samples/` — папка с примерами CSV-файлов, на основе которых создаются новые данные.
- **Документация**:
  - `README.md` — краткая инструкция.
  - `LICENSE` — лицензионное соглашение (MIT).

---

## 2. Порядок работы

### Шаг 1: Подготовка данных
Поместите ваши образцы CSV-файлов в папку `data/samples/base/` (для генерации новых данных) или `data/samples/dimensional/` (для изменения существующих).

### Шаг 2: Настройка конфигурации
Откройте `config.yaml` в любом текстовом редакторе (Блокнот, VS Code и т.д.) и укажите:
- Имя выходного файла.
- Количество строк для генерации (`rows`).
- Файл-образец (`sample_file`).
- Типы аномалий, если они нужны.

### Шаг 3: Запуск программы

#### В Windows:
1. Зайдите в папку с программой.
2. Удерживая `Shift`, нажмите правую кнопку мыши и выберите "Открыть окно PowerShell здесь" или "Открыть в терминале".
3. Введите команду:
   ```cmd
   .\datagen.exe -config config.yaml
   ```

#### В macOS / Linux:
1. Откройте Терминал и перейдите в папку с программой.
2. Дайте права на выполнение (только при первом запуске):
   ```bash
   chmod +x datagen_macos  # или datagen_linux
   ```
3. Запустите программу:
   ```bash
   ./datagen_macos -config config.yaml
   ```

#### В Google Colab:
Загрузите архив, распакуйте его и выполните в ячейке:
```python
!chmod +x datagen_linux
!./datagen_linux -config config.yaml
```

## 3. Проверка результатов
Сгенерированные данные появятся в папке `data/generated/` (директория настраивается в `config.yaml`).

## 4. Справка и поддержка
Для вызова встроенной справки используйте флаг `--help`:
```bash
./datagen_macos --help
```
